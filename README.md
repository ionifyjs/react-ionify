# React Ionify

A demo React app powered by [Ionify](https://www.npmjs.com/package/@ionify/ionify).

This project showcases Ionify dev/build workflow with React Router, state management, env/define replacements, and dependency optimization behavior in a realistic app setup.

## Links

- Ionify npm package: [@ionify/ionify](https://www.npmjs.com/package/@ionify/ionify)
- Ionify website: [ionify.cloud](https://ionify.cloud/)
- Ionify public repo: [ionifyjs/ionify](http://github.com/ionifyjs/ionify)
- This demo repo: [ionifyjs/react-ionify](https://github.com/ionifyjs/react-ionify)
